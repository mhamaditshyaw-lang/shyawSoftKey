# User Management Quick Guide

## Where to Access User Management

### 1. User Management Page (Full Features)
- **Location**: Click "User Management" in the left sidebar navigation
- **Access**: Admin and Manager roles only
- **Features**: 
  - View all users in a table
  - Add new users with detailed permissions
  - Edit user information and roles
  - Set specific permissions for each user
  - Delete users (admin only)
  - Search and filter users by role

### 2. Employee Management Page (Basic Employee Info)
- **Location**: Click "Employee Management" in the left sidebar navigation
- **Access**: Admin and Manager roles
- **Features**: Basic employee information management

## Available User Roles

1. **Admin** - Full system access
2. **Manager** - User management access + reports
3. **Security** - Basic access to tasks and reviews
4. **Office** - Limited permissions (configurable)

## How to Add Office Manager

### Option 1: Using User Management Page
1. Go to "User Management" in the sidebar
2. Click "Add User" button
3. Fill in the form and select "Office" role
4. Click "Settings" icon next to the user to configure permissions
5. Enable permissions like "View Reports", "Edit Users", etc.

### Option 2: Using Add Employee Page  
1. Go to "Employee Management" in the sidebar
2. Click "Add Employee" 
3. Fill in the form and select "Office" from the role dropdown
4. The office role is now available in the dropdown

## Permission System

Each user role can have customizable permissions:
- View Users, Edit Users, Delete Users
- View Todo Lists, Edit Todo Lists
- View Interviews, Edit Interviews
- View Reports, Manage Notifications

Use the User Management page to configure these permissions for each user.
